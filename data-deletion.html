User Data Deletion Instructions

Last updated: July 2026

If you would like your data removed from CreatorOps, please follow the instructions below.

How to Request Deletion

  1. Send an email to learnwithdevopsengineer3682@gmail.com
  2. Use the subject: CreatorOps Data Deletion Request
  3. Include your Instagram username so we can identify your records.

Please do not send your Instagram password or Meta access token.

What We Delete

Upon verification we will delete, where applicable:

  • Stored Instagram account identifiers
  • Automation configurations
  • Webhook events
  • Message delivery records
  • Associated application data

Processing Time

Deletion requests are normally completed within 30 days.

Disconnecting the Application

Removing CreatorOps from your Meta or Instagram account prevents future access but does not automatically remove previously stored data. If you also want stored data deleted, please submit a deletion request using the email above.
